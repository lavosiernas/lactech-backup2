Introduction

SafeCode is a professional, high-performance IDE built for speed and aesthetics. With a monochrome design philosophy and native performance, it provides everything you need for modern development without the bloat.

Version 1.0.0

This documentation covers SafeCode version 1.0.0 and above.

Why SafeCode?

  • Minimalist Design - Clean, distraction-free interface that puts your code first
  • Native Performance - Built with Electron for fast, responsive editing
  • Monaco Editor - Industry-standard editor with IntelliSense and syntax highlighting
  • Extensible - Comprehensive API for building custom extensions

Installation

SafeCode can be installed on Windows, macOS, and Linux. Choose your platform below:

Windows

Install SafeCode using PowerShell:

powershell -ExecutionPolicy ByPass -c "irm https://safenode.cloud/safenode/safecode/install.ps1 | iex"

macOS / Linux

Install using the install script:

curl -fsSL https://safenode.cloud/safenode/safecode/install.sh | bash

Requirements

  • Node.js 16 or higher
  • Git (for version control features)
  • Electron (installed automatically)
Note

Make sure you have Node.js installed before running the installation command.

Quick Start

Get up and running with SafeCode in minutes.

1. Launch SafeCode

After installation, launch SafeCode from your applications menu or command line:

safecode

2. Open a Project

Open a folder or create a new file:

  • Ctrl+O - Open folder
  • Ctrl+N - New file

3. Start Coding

SafeCode provides IntelliSense, syntax highlighting, and code completion out of the box for 100+ languages.

Editor

The SafeCode editor is powered by Monaco, the same editor that powers VS Code.

Features

  • IntelliSense - Intelligent code completion
  • Syntax Highlighting - Support for 100+ languages
  • Multi-cursor Editing - Edit multiple locations simultaneously
  • Code Folding - Collapse code blocks for better readability
  • Find and Replace - Powerful search with regex support

Split View

Work with multiple files side by side:

  • Ctrl+\\ - Split editor vertically
  • Ctrl+K Ctrl+\\ - Split editor horizontally

Terminal

SafeCode includes an integrated terminal with PTY support and custom themes.

Opening the Terminal

  • Ctrl+` - Toggle terminal
  • Ctrl+Shift+` - Create new terminal

Features

  • Multiple terminal instances
  • Split terminal view
  • Custom shell integration
  • Monochrome theme matching the editor

Git Integration

SafeCode provides built-in Git support for version control.

Features

  • Visual Diff - See changes inline in the editor
  • Commit Management - Stage and commit changes
  • Branch Management - Create, switch, and merge branches
  • Conflict Resolution - Visual merge conflict resolution

Common Operations

# Stage all changes git add . # Commit with message git commit -m "Your message" # Push to remote git push origin main

Extensions

Extend SafeCode's functionality with custom extensions.

Installing Extensions

Extensions are installed in the extensions directory:

~/.safecode/extensions/

Popular Extensions

  • Core Intelligence - AI-powered code completion
  • Theme Pack - Additional color themes
  • Snippets - Code snippet library

Settings

Customize SafeCode to match your workflow.

Accessing Settings

  • Ctrl+, - Open settings

Common Settings

{ "editor.fontSize": 14, "editor.tabSize": 2, "editor.wordWrap": "on", "theme": "monochrome-true-dark", "terminal.shell": "powershell" }

Themes

SafeCode comes with a carefully crafted monochrome theme system.

Available Themes

  • Monochrome True Dark - Pure black background
  • Monochrome Dark - Dark gray background
  • Monochrome Light - Light theme variant

Changing Themes

Change themes from the command palette:

  • Ctrl+Shift+P - Open command palette
  • Type "theme" and select your preferred theme

Extension API

Build powerful extensions for SafeCode using our comprehensive API.

Creating an Extension

Extensions are Node.js modules with a specific structure:

my-extension/ ├── extension.js ├── package.json └── README.md

Basic Extension

// extension.js module.exports = { activate(context) { console.log('Extension activated!'); // Register commands, add UI elements, etc. }, deactivate() { console.log('Extension deactivated'); } };

API Reference

The SafeCode API provides access to:

  • Editor manipulation
  • File system operations
  • Terminal integration
  • UI customization
  • Command registration
