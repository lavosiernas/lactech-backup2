Desenvolvedores

Existem algumas maneiras de desenvolvedores interagirem com o Mapas Culturais. A primeira é através da nossa API. Com ela você pode acessar os dados públicos no nosso banco de dados e utilizá-los para desenvolver aplicações externas. Além disso, o Mapas Culturais é construído a partir do software livre Mapas Culturais, criado em parceria com o Instituto TIM.
