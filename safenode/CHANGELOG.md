# Changelog - SafeCode IDE

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-08

### Added
- **Precision Landing Page**: Ultra-premium minimalist design with technical grid background and floating "Bridge" navigation.
- **Monochrome IDE Core**: Fully professional black, white, and gray interface optimized for long coding sessions.
- **Settings & Preferences**: Persistent local storage for font size, tab size, word wrap, and UI behavior.
- **IntelliSense V2**: Enhanced Monaco Editor configuration with parameter hints, advanced autocomplete, and detailed suggestions.
- **Extensions Gallery**: Searchable sidebar panel for plugins (simulated for v1.0).
- **Live Preview System**: Real-time rendering with mobile (iOS/Android) and tablet frame simulation.
- **Integrated Terminal**: Fully functional xterm.js terminal with multi-instance support.
- **Project Explorer**: Advanced file tree with folder expansion and file type icons.
- **Command Palette**: Centralized search for commands and rapid IDE interaction.
- **Update System**: Integrated version checking for semi-automatic updates.

### Fixed
- Header compression issue on large displays.
- Visual consistency across buttons and sidebar components.
- Glassmorphism balance to ensure clarity and premium feel.
