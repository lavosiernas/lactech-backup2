Installation

Clone the repository and start the development server.

1

Clone Repository

Clone SafeCode to your machine.

Terminal 
git clone https://github.com/safenode/safecode.git
cd safecode/safecode
2

Install Dependencies

Install all required packages.

Terminal 
npm install
3

Start Server

Launch the development server.

Terminal 
npm run dev

Requirements

  • Node.js — Version 18+
  • npm — or yarn
  • Git — For version control
  • Modern Browser — Chrome, Firefox, Safari, Edge

Quick Start

Get running in under 5 minutes.

Launch SafeCode

After starting the server, open your browser:

URL 
http://localhost:8080

Open a Project

  • Ctrl + Shift + O — Open folder
  • Ctrl + N — New file
  • Ctrl + O — Open file
Pro Tip

Press Ctrl + Shift + P to open the command palette.

Editor

IntelliSense

Syntax Highlighting

Find & Replace

Split View

  • Ctrl + \ — Split vertically
  • Ctrl + K Ctrl + \ — Split horizontally

Terminal

Integrated terminal powered by xterm.js.

Shortcuts

  • Ctrl + ` — Toggle terminal
  • Ctrl + Shift + ` — New terminal

Features

  • Multiple Instances — Run several terminals
  • Split Terminal — Divide into panes
  • Shell Integration — Bash, zsh, PowerShell
  • Theme Sync — Matches editor theme

Git Integration

Built-in version control for seamless collaboration.

Features

  • Visual Diff — See changes inline
  • Commit — Stage and commit with UI
  • Branches — Create, switch, merge

Settings

Customize SafeCode to your workflow.

Access Settings

  • Ctrl + , — Open settings
  • Ctrl + Shift + P — Type "settings"
JSON 
{
  "editor.fontSize": 14,
  "editor.tabSize": 2,
  "editor.wordWrap": "on",
  "editor.minimap.enabled": true,
  "theme": "monochrome-true-dark"
}
