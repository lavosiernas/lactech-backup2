Introduction

SafeCode is a modern, web-based IDE built with React, TypeScript, and Vite. With a minimalist design philosophy and powerful features, it provides everything you need for modern web and mobile development without the bloat.

Version 2.0.0

This documentation covers SafeCode version 2.0.0 and above.

Why SafeCode?

  • Minimalist Design - Clean, distraction-free interface that puts your code first
  • Modern Stack - Built with React, TypeScript, and Vite for fast, responsive editing
  • Monaco Editor - Industry-standard editor with IntelliSense and syntax highlighting
  • Live Preview - Real-time preview with iOS and Android device simulation
  • Web-Based - Runs entirely in the browser, no installation required

Installation

SafeCode runs entirely in the browser. Simply clone the repository and start the development server.

Clone Repository

git clone https://github.com/safenode/safecode.git cd safecode/safecode

Install Dependencies

npm install

Run Development Server

npm run dev

Requirements

  • Node.js 18 or higher
  • npm or yarn
  • Git (for version control features)
Note

Make sure you have Node.js installed before running the installation commands.

Quick Start

Get up and running with SafeCode in minutes.

1. Launch SafeCode

After starting the development server, open your browser and navigate to:

http://localhost:8080

2. Open a Project

Open a folder or create a new file:

  • Ctrl+Shift+O - Open folder
  • Ctrl+N - New file

3. Start Coding

SafeCode provides IntelliSense, syntax highlighting, and code completion out of the box for 100+ languages.

Editor

The SafeCode editor is powered by Monaco, the same editor that powers VS Code.

Features

  • IntelliSense - Intelligent code completion
  • Syntax Highlighting - Support for 100+ languages
  • Multi-cursor Editing - Edit multiple locations simultaneously
  • Code Folding - Collapse code blocks for better readability
  • Find and Replace - Powerful search with regex support

Split View

Work with multiple files side by side:

  • Ctrl+\\ - Split editor vertically
  • Ctrl+K Ctrl+\\ - Split editor horizontally

Terminal

SafeCode includes an integrated terminal powered by xterm.js.

Opening the Terminal

  • Ctrl+` - Toggle terminal
  • Ctrl+Shift+` - Create new terminal

Features

  • Multiple terminal instances
  • Split terminal view
  • Custom shell integration
  • Monochrome theme matching the editor

Git Integration

SafeCode provides built-in Git support for version control.

Features

  • Visual Diff - See changes inline in the editor
  • Commit Management - Stage and commit changes
  • Branch Management - Create, switch, and merge branches
  • Clone Repository - Clone repositories directly from the IDE

Common Operations

# Stage all changes git add . # Commit with message git commit -m "Your message" # Push to remote git push origin main

Live Preview

SafeCode features an advanced live preview system with iOS and Android device simulation.

Features

  • Device Simulation - Preview your code on iOS and Android devices with realistic lock screens and home screens
  • Real-time Updates - See your changes instantly as you code
  • Browser Integration - Test with Safari (iOS) and Chrome (Android) browsers
  • Device Controls - Rotate, lock, power, and navigate between screens
  • Multiple Modes - Desktop, tablet, and mobile preview modes

Usage

Open the Live Preview panel and select your preferred device mode. The preview updates automatically as you edit your code.

Settings

Customize SafeCode to match your workflow.

Accessing Settings

  • Ctrl+, - Open settings

Common Settings

{ "editor.fontSize": 14, "editor.tabSize": 2, "editor.wordWrap": "on", "theme": "monochrome-true-dark", "terminal.shell": "powershell" }

Themes

SafeCode comes with a carefully crafted monochrome theme system.

Available Themes

  • Monochrome True Dark - Pure black background
  • Monochrome Dark - Dark gray background
  • Monochrome Light - Light theme variant

Changing Themes

Change themes from the command palette:

  • Ctrl+Shift+P - Open command palette
  • Type "theme" and select your preferred theme
